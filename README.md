# AtlasCompleteness
This project integrates single-cell RNA sequencing (scRNA-seq) data with statistical biodiversity estimation methods to quantify cellular diversity within organs and assess the completeness of large-scale sequencing efforts.
## Chao2 Saturation Wrapper

Wrapper for estimating atlas completeness and diversity saturation using Chao2 statistics.

Location:

```text
chao2_wrapper/
```

Main script:

```text
chao2_wrapper/chao2_saturation.py
```

Documentation:

```text
chao2_wrapper/README.md
```

Example import:

```python
from chao2_wrapper.chao2_saturation import chao2_saturation_wrapper
```
